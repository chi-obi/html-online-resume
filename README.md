# html-online-resume
The goal was to create a comprehensive multi-page website employing proper HTML structure and CSS Styling.
Website Structure: 
 -Organized content into multipage web pages i.e Hobbies, Contact Me. 
- Established interlinking between different pages effectively using anchor tags "<a>" and the "href" element. 
HTML Fundamentals: 
- Built a solid basis with HTML tags and linking pages - using navigation menus for seamless transitions between pages. 
- Understand basic website development principles and best practices for a responsive and user-friendly design.
CSS for Design:
- Used CSS to style pages consistently and create a cohesive look across all pages.
Learnin Outcomes:
- The importance of planning and structuring a website, as well as the practical application of HTML, CSS and Javascript.
